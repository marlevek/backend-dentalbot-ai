Política de Privacidade – CoderTec

Última atualização: 27/11/2025

1. Introdução

A CoderTec desenvolve soluções de automação, chatbots, plataformas web e integrações com IA. Esta política explica como tratamos dados em nossos sistemas, incluindo o sistema Dentalbot-AI.

2. Coleta de Dados

Coletamos apenas dados fornecidos voluntariamente pelo usuário ou pela empresa contratante, incluindo:

  • Nome e contato do cliente
  • Mensagens enviadas ou recebidas pelo WhatsApp Business API
  • Dados necessários para funcionamento do sistema

3. Uso dos Dados

Utilizamos os dados exclusivamente para:

  • Envio e recebimento de mensagens
  • Atendimento automatizado
  • Melhorias nos sistemas CoderTec

4. Armazenamento e Segurança

Adotamos boas práticas de segurança e não compartilhamos dados com terceiros sem permissão.

5. Compartilhamento de Dados

Nunca vendemos ou compartilhamos dados pessoais. O uso da API do WhatsApp segue as regras da Meta.

6. Direitos do Usuário

O usuário pode solicitar exclusão, alteração ou cópia dos dados armazenados.
